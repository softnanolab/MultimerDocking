## Installation

With pip and conda venv:
```
git clone https://github.com/softnanolab/MultimerDocking.git
cd MultimerDocking
mamba create -n dock python=3.11
mamba activate dock
pip install -e ".[dev]"
```

With uv:
```
git clone https://github.com/softnanolab/MultimerDocking.git
cd MultimerDocking
uv sync
```
